1. Two Sum

Given an array of integers nums and an integer target, return the indices of the two numbers such that they add up to target.


You may assume that each input would have exactly one solution, and you may not use the same element twice.


You can return the answer in any order.

Example 1
Input:  nums = [2, 7, 11, 15], target = 9
Output: [0, 1]
Why:    nums[0] + nums[1] = 2 + 7 = 9 ‚úì
Example 2
Input:  nums = [3, 2, 4], target = 6
Output: [1, 2]
Why:    nums[1] + nums[2] = 2 + 4 = 6 ‚úì
Example 3
Input:  nums = [3, 3], target = 6
Output: [0, 1]

Progressive Hints

The simplest approach: use two loops. For every element nums[i], check every other element nums[j] to see if nums[i] + nums[j] == target. This works but takes O(n¬≤) time. Can you do better?
For each number, you need to find its "complement" ‚Äî the number that when added to it equals the target. If the number is 2 and target is 9, the complement is 7. How can you quickly check if a complement exists?
Use a Hash Map (dictionary in Python, object in JavaScript). As you loop through the array, store each number and its index in the hash map. Before adding a number, check if its complement is already in the map. This gives you O(1) lookups!
1. Create an empty hash map: seen = {}
2. Loop through each number with its index (i, num)
3. Calculate complement = target - num
4. If complement is in seen ‚Üí return [seen[complement], i]
5. Otherwise ‚Üí add seen[num] = i and continue
Time: O(n) ¬∑ Space: O(n)

Time Complexity
O(n)
Space Complexity
O(n)
# Python Solution ‚Äî Hash Map Approach def twoSum(nums, target): seen = {} # num ‚Üí index for i, num in enumerate(nums): complement = target - num if complement in seen: return [seen[complement], i] seen[num] = i return [] # no solution

üß† Plain English Explanation

1
We create a "phone book" (hash map) where we store numbers we've already seen, along with their positions in the array.
2
For each number, we ask: "Is the number I need to pair with (the complement) already in my phone book?" For example, if target=9 and current number=2, we look for 7.
3
If YES ‚Üí we found our answer! Return the index of the complement (from the phone book) and the current index.
4
If NO ‚Üí add the current number to the phone book and move to the next number. We only need ONE loop, making this O(n)!
